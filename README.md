# everybody is in the place 

Quickly setup/fix my environment.

## Requirements

Install [Homebrew](http://brew.sh)

```bash
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## Execution

```
alias eiitp=make
eiitp lets
eiitp go
```
